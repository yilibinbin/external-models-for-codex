# Claude for Codex Native Review Pack

This directory is an optional Claude Code plugin bundle exported by Claude for Codex.

Use it only when you explicitly want Claude Code to load the native review agents:

```bash
claude --plugin-dir /path/to/claude-for-codex/claude-code-plugin
```

The bundle contains only read-only review agents. It intentionally does not include Codex skills, hooks, MCP servers, settings, shell commands, or write-tool grants. The Codex plugin root is not a Claude Code plugin root.
